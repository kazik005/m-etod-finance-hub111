# Инструкция по установке и развертыванию M-etod Finance Hub

M-etod Finance Hub построен на современном стеке технологий: React + Vite + Tailwind CSS + Blink SDK.

## Локальная разработка

1. Убедитесь, что у вас установлен Node.js и Bun (рекомендуется).
2. Установите зависимости:
   ```bash
   bun install
   ```
3. Запустите проект:
   ```bash
   bun run dev
   ```

## Настройка окружения (Secrets)

Для работы SDK необходимо добавить следующие переменные в настройки вашего хостинга или файл `.env`:

- `VITE_BLINK_PROJECT_ID`: ID вашего проекта в Blink.
- `VITE_BLINK_PUBLISHABLE_KEY`: Публичный ключ API.

## Первичная настройка сайта

1. Авторизуйтесь на сайте под своим email.
2. Чтобы стать администратором, убедитесь, что ваш email совпадает с `admin@m-etod.ru` или измените логику в `src/contexts/AuthContext.tsx`.
3. Перейдите в раздел **Админ** (появится в меню после входа).
4. На вкладке **Обзор** нажмите кнопку **"Заполнить демо-данными"**. Это создаст начальные категории, статьи и офферы.

## Развертывание на хостинг

### Вариант 1: Blink (Автоматически)
Сайт уже развернут и доступен по адресу вашего проекта `{project-id}.live.blink.new`.

### Вариант 2: Статический хостинг (Vercel, Netlify, Cloudflare Pages)
1. Соберите проект:
   ```bash
   bun run build
   ```
2. Загрузите содержимое папки `dist` на ваш хостинг.
3. Настройте "Fallback" для всех маршрутов на `index.html` (SPA routing). В проекте уже есть файл `public/_redirects` для Netlify.

### Вариант 3: Собственный сервер
Используйте любой веб-сервер (Nginx, Apache) для раздачи статических файлов из папки `dist`.

## Администрирование
- **Категории**: Создавайте разделы для офферов, статей и форума.
- **Офферы**: Добавляйте финансовые предложения с внешними партнерскими ссылками.
- **Статьи**: Пишите уникальный контент в визуальном редакторе.
- **Форум**: Модерируйте темы и сообщения пользователей.
- **Курсы валют**: Обновляйте курсы вручную для отображения актуальной информации.
